Mercado Pago
Credenciales de integración
● No configurado
ℹ️Obtené tus credenciales en mercadopago.com.ar → Tus integraciones → Credenciales.
Se usa en el frontend para mostrar el formulario de pago
Secreto — solo se usa en el servidor backend
Flujo de Pago
1
Cliente completa el formulario
Datos de envío y selecciona Mercado Pago como medio de pago
2
Se crea la preferencia de pago
El sistema llama a POST /checkout/preferences con el Access Token
3
Redirección a Mercado Pago
Cliente paga con tarjeta, transferencia o billetera en la plataforma segura de MP
4
Webhook confirma el pago
MP notifica tu servidor en /api/mp-webhook y el pedido se marca Pagado
5
Pedido confirmado ✅
Email de confirmación al cliente, stock se actualiza automáticamente
Código Backend Node.js
const { MercadoPagoConfig, Preference, Payment } = require('mercadopago');
const client = new MercadoPagoConfig({ accessToken: process.env.MP_ACCESS_TOKEN });

// Crear preferencia de pago
app.post('/api/create-preference', async (req, res) => {
  const { items, customer, orderId } = req.body;
  const preference = new Preference(client);
  const result = await preference.create({ body: {
    items: items.map(i => ({ title: i.name, quantity: 1, unit_price: i.price, currency_id: 'ARS' })),
    payer: { name: customer.name, email: customer.email },
    back_urls: { success: 'https://tu-tienda.com/pago/exito', failure: 'https://tu-tienda.com/pago/error' },
    auto_return: 'approved',
    notification_url: 'https://tu-tienda.com/api/mp-webhook',
    external_reference: orderId,
  }});
  res.json({ init_point: result.init_point });
});

// Webhook
app.post('/api/mp-webhook', async (req, res) => {
  const { type, data } = req.body;
  if (type === 'payment') {
    const paymentData = await new Payment(client).get({ id: data.id });
    if (paymentData.status === 'approved') {
      await db.orders.update({ id: paymentData.external_reference }, { payStatus: 'Pagado', status: 'Preparando' });
    }
  }
  res.sendStatus(200);
});
